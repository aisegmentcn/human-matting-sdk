# human-matting-sdk 立志于做全球效果最好/性价比最优的抠像SDK
本项目包括Windows/Android/iOS三个平台的视频人像分割SDK和图片人像抠图matting SDK。   
SDK包括免费版和收费版。免费版对输出图片/视频分辨率有一定限制，且只输出合成图，不输出alpha图。其他跟收费版完全相同。
